## 繁體中文說明

**iMyLoc** 是一個專為 iOS 裝置設計的虛擬定位工具，無需越獄（Jailbreak）即可實現全域級別的 GPS 定位修改。此工具透過電腦與 iOS 裝置進行本地通訊，提供安全且穩定的定位模擬服務。

### 軟體特色功能

1. **虛擬定位 (GPS Spoofing)**
   直接在地圖上點擊您想要去的地方，瞬間將 iOS 裝置的 GPS 修改至世界任何一個角落。支援即時切換目標地，快速且精準。
2. **自動走路模式 (Auto Route Walking)**
   支援「自訂目的地」導航以及「產生隨機路線」兩種模式。系統會沿著道路自動規劃移動軌跡，您還可以自訂移動速度（如：步行、單車、開車等），讓 GPS 移動軌跡看起來自然且真實。
3. **遙桿行走模式 (Joystick Mode)**
   內建即時螢幕遙桿（Joystick），讓您可以像玩遊戲一樣，360 度手動控制 iOS 定位的移動方向。非常適合需要精細走位或是微調座標的使用情境！
4. **免費免安裝綠色軟體 (Free & Portable)**
   本軟體完全免費且無需安裝，下載即可直接使用。不會在系統留下多餘痕跡，帶給您最乾淨、輕量的使用體驗。

### 系統前置作業 (必讀)

由於蘋果安全機制的限制，在初次使用 iMyLoc 之前，請務必完成以下兩項準備工作：

1. **安裝 iTunes for Windows (必備驅動)**
   本軟體依賴蘋果官方的 USB 底層通訊協定（Apple Mobile Device Service）。請務必前往[蘋果官網下載並安裝「傳統桌面版本」的 iTunes](https://www.apple.com/itunes/download/win64)。安裝完成且手機接上電腦時，**請在手機上點選「信任這部電腦」**。
2. **開啟 iOS 開發者模式 (Developer Mode)**
   對於 iOS 16 以上版本的裝置，iOS 預設會隱藏「開發者模式」選項，必須手動觸發。您可以使用 Sideloadly 來開啟：
   * 下載並安裝 **[Sideloadly](https://sideloadly.io/)**。
   * 將手機接上電腦，開啟 Sideloadly 並輸入您的 Apple ID。
   * 將任意一個 `.ipa` 檔案（應用程式安裝檔）拖曳至 Sideloadly 視窗中，點擊 `Start` 進行安裝。若無現成的檔案，可隨意於網路上下載一個免費的 `.ipa` 檔。
   * 安裝完成後，請至手機的 `設定 -> 一般 -> VPN與裝置管理`，信任您的 Apple ID 憑證。
   * 接著至手機的 `設定 -> 隱私權與安全性 -> 開發者模式`，將其打開。
   * 手機會提示需要重啟。重啟完成並解鎖螢幕後，在彈出的確認視窗點擊「開啟」並輸入螢幕密碼即可完成。

### Prerequisites & Setup (Required)

Due to Apple's security policies, you must fulfill the following two environmental requirements before using iMyLoc for the first time:

1. **Install iTunes for Windows (Required Drivers)**
   This software heavily depends on the Apple Mobile Device Service via USB. Please [download and install the traditional Desktop Version of iTunes directly from Apple's Website](https://www.apple.com/itunes/download/win64). Once installed, connect your device to your computer via USB and explicitly tap **"Trust This Computer"** on your iPhone screen.
2. **Enable iOS Developer Mode**
   For iPhones running iOS 16 and above, the "Developer Mode" option is hidden by default. You can use Sideloadly to trigger and enable it:
   * Download and install **[Sideloadly](https://sideloadly.io/)**.
   * Connect your phone to your PC, open Sideloadly, and enter your Apple ID.
   * Drag and drop any `.ipa` file (iOS app package) into the Sideloadly window and click `Start` to sideload it. If you don't have one, you can download any basic or free `.ipa` file from the web.
   * Once sideloaded, go to your iPhone's `Settings -> General -> VPN & Device Management` and trust your Apple ID certificate.
   * Then, navigate to `Settings -> Privacy & Security -> Developer Mode` and toggle it on.
   * Your phone will prompt a restart. After it reboots and you unlock it, tap "Turn On" in the pop-up prompt and enter your device passcode.
